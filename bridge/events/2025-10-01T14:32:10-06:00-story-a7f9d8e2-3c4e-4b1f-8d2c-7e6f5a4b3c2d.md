## Story

The command standardization pattern emerged from the AAFC Herbarium project when we realized we were reinventing multi-agent coordination wheels in each new
project. Every project needed deployment orchestration, agent creation, session handoffs, and bridge synchronization - but we were rebuilding these from scratch
each time.

By extracting 5 battle-tested commands (/deploy, /bridge-agent-create, /session-handoff, /sync-with-native, /bridge-extraction-prep) and creating a template
pattern with automatic path resolution, we transformed project setup from days of manual configuration to 2 hours of guided integration.

The pattern proved valuable because it encoded proven workflows while remaining simple enough for rapid adoption. The key insight: standardize infrastructure,
customize domain logic. This allowed projects to inherit production-grade coordination while maintaining their unique requirements.
