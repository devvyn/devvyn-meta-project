# Bibliography

**Overview**: Citation network of books, papers, and research organized by intellectual domain and connection patterns.

## Navigation Map

This bibliography is organized around **citation neighbors**: works that are frequently cited together, reference each other, or address similar questions from complementary perspectives.

### Deep Work Network

- [Deep Work](deep-work-network/deep-work-newport.md) - Cal Newport (2016)

## Reading Sequences

### For Understanding Behavior Change
1. **Atomic Habits** (Clear) - Most actionable, accessible entry point
2. **Tiny Habits** (Fogg) - Scientific foundation, practical method
3. **Nudge** (Thaler/Sunstein) - Choice architecture theory
4. **Hooked** (Eyal) - Product design perspective

### For Mastering Attention
1. **Deep Work** (Newport) - Core framework
2. **Flow** (Csikszentmihalyi) - Theoretical foundation
3. **Attention Span** (Mark) - Contemporary research
4. **Rapt** (Gallagher) - Philosophical perspective

### For Environmental Design
1. **The 99% Invisible City** (Mars/Kohlstedt) - Accessible, engaging
2. **A Pattern Language** (Alexander) - Comprehensive reference
3. **The Timeless Way of Building** (Alexander) - Philosophical depth
4. **The Experience of Nature** (Kaplan) - Restorative environments
